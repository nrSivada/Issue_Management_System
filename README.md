# Issue Management System

## Features
-  create and manage issues
-  view a list of issues
-  comment system
-  searching and filtering
-  issue analysis using Gemini
-  swagger API documentation

Frontend:
- React
- TypeScript
- Tailwind CSS
Backend:
- Node.js
- TypeScript
- Express
- PostgreSQL
- Drizzle ORM
AI:
-Google Gemini
